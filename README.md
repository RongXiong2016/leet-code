## 归纳总结
   - 遇到树相关的优先考虑递归
   - DP相关的，抓住状态转移方程即 找规律
   - 需要额外数据结构时，可以想想栈stack
   - 回文topic相关的 优先递归
   - 二进制的相关技巧
       - i&1：奇数等于1，偶数等于0
       - n&(n-1)就是把n的最未的1变成0
       - ^ 异或。相同为0，不同为1
           - x^x=0
           - x^0=x
           - a^b^b = a^0 = a
       - 一个数除以2  >>1 更高效
       - x&x-1 ==0 判断x是否是2的n次方
   ### 数组相关
   - 题目中带有sorted的，优先想到二分法例如 LeetCode34 题
   - 题目中包含2倍相关的，优先想到是否可以使用快慢指针 例如Leetcode1470
